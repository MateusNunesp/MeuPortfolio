# 🌐 Meu Portfólio Pessoal

Bem-vindo ao repositório do meu portfólio! 

Este projeto foi desenvolvido do zero com o objetivo principal de colocar em prática e consolidar meus conhecimentos iniciais em **Desenvolvimento Web Front-End**. Ele serve tanto como uma vitrine para os meus projetos (como o Clone do Termo) quanto como um laboratório de estudos.


---

## 🎯 Objetivo do Projeto

A ideia principal ao construir este portfólio não foi apenas ter uma página online, mas sim entender na prática como a web funciona sem o uso de frameworks pesados. O foco do desenvolvimento foi:
* Estruturar o conteúdo de forma semântica.
* Criar um layout atraente e responsivo.
* Adicionar interatividade básica e manipulação do DOM.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando os pilares clássicos da web (Vanilla):

* **HTML5:** Estruturação semântica, organização de seções (Sobre mim, Projetos, Contato) e acessibilidade básica.
* **CSS3:** Estilização global, uso de Flexbox/Grid para alinhamento e Media Queries para garantir que o site funcione bem em celulares e computadores.
* **JavaScript:** Adição de interatividade e dinamismo à página.

---

## 📚 O que eu aprendi

Durante a construção deste portfólio, pude aplicar na prática conceitos como:
* **Responsividade:** Adaptar o layout para diferentes tamanhos de tela.
* **Organização de Código:** Separar as responsabilidades entre estrutura (HTML), estilo (CSS) e comportamento (JS).
* **Navegação:** Criar links âncora e estruturar a experiência do usuário.
